# 😂 Malayalam Meme Extension for VS Code

> Add some *Malayalam vibe* to your coding experience 🇮🇳
> Because sometimes… your code deserves a reaction 😭

---

## 🔥 What is this?

**Malayalam Meme Extension** is a fun, lightweight VS Code extension that reacts to your coding activity with:

* 😂 Malayalam meme texts
* 🔊 Audio alerts
* 🎬 GIF / animation popups
* 🐾 Interactive dev companion (optional)

Turn your coding session into something **fun, expressive, and slightly chaotic**.

---

## 🎯 Features

### 🎭 Context-Aware Reactions

* ❌ Errors → Funny Malayalam roasts
* ✅ Success → Celebration reactions
* ⏳ Inactivity → Wake-up alerts
* 🚀 Actions → Smart responses

---

### 🎲 Randomized Alerts

* No repetitive messages
* Each event triggers a **different meme**
* Pack-based randomization

---

### 🔊 Audio + 🎬 Visual Feedback

* Meme sound effects
* GIF / animation popups
* Synced reactions for better experience

---

### 🐾 Optional Animated Companion

* A small character that:

  * Reacts to your coding
  * Shows emotions 😭😡😎
  * Moves inside the editor

---

### 📦 Downloadable Meme Packs

* Supports custom packs

### 🌍 Community Support

## 🚀 Installation

### From VS Code Marketplace

1. Open Extensions (`Ctrl + Shift + X`)
2. Search for **Malayalam Meme Extension**
3. Click Install

---

### Manual Install

```bash
code --install-extension malayalam-meme-extension.vsix
```

---

## 📦 Meme Packs

### Install from URL

1. Open Command Palette (`Ctrl + Shift + P`)
2. Run:

```
Install Meme Pack from URL
```

---

### Import Pack

```
Import Meme Pack (ZIP)
```

---

### Pack Format

```json
{
  "name": "Malayalam Pack",
  "version": "1.0",
  "events": {
    "error": [
      {
        "text": "Entha mone ee code 😭",
        "audio": "error.mp3",
        "gif": "error.gif"
      }
    ]
  }
}
```
